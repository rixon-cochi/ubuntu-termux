# ubuntu-termux

Ubuntu in your termux application without a rooted device

## Installation Setup

* ``` apt-get update && apt-get upgrade -y ```

* ``` apt-get install git -y ```

* ``` git clone https://github.com/rixon-cochi/ubuntu-termux.git ```

* ``` cd ubuntu-termux ```

* ``` bash run.sh ```

* ``` bash finish ```

* ``` ./startubuntu.sh ```
